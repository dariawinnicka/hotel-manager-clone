# Hotel Manager Clone

This repository contains a near-exact clone of the [Hotel Manager](http://www.hotelmanager.co) website, reflecting its state in 2024. The project focuses on replicating the front-end design and user experience of the original platform.

## Technologies

- **HTML**: For the structure and layout of the web page.
- **CSS**: For styling and making the application visually appealing.
- **JavaScript**: For interactivity and dynamic functionality.
- **Font Awesome**: For incorporating social media icons.
- **CSS.gg**: For incorporating key icon.
- **Google Fonts**: For incorporating fonts and tick icons.
